# Architecture Decision Records

Short, opinionated records of decisions that shaped this codebase. Each ADR
captures the context at the time, the decision made, and the tradeoffs we
accepted — so future contributors can tell a load-bearing choice from an
accident.

Numbering is sequential. Status is one of `Proposed`, `Accepted`,
`Superseded by #NNNN`, `Deprecated`.
